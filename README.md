Descripción del proyecto

Este es un programa CLI (Command Line Interface) diseñado para calcular interés compuesto de forma rápida, clara y sin depender de herramientas web limitadas o de pago. Lo hice porque, al necesitar calcularlo, descubrí que casi no había opciones gratuitas que fueran prácticas y completas. Así que desarrollé una herramienta ligera, intuitiva y útil para cualquiera que quiera entender mejor cómo crece su dinero.

El programa no solo calcula el monto final, también muestra datos clave que normalmente las calculadoras básicas no dan:

Total ahorrado: todo el dinero que aportaste durante el tiempo definido.

Total generado por intereses: cuánto ganaste realmente gracias al interés compuesto.

Estos datos permiten entender mejor cómo evoluciona una inversión o un ahorro, y ver con claridad la diferencia entre lo que aportas y lo que ganas.

¿Por qué usar este programa?

Es rápido: solo escribes, confirmas y tienes el resultado.

Es simple: no hace falta abrir páginas web ni navegar entre anuncios.

Es preciso: calcula valores clave que otras herramientas omiten.

Es ideal para terminal: perfecto si trabajas en entornos CLI y quieres eficiencia.

Es totalmente gratuito y local: no dependes de conexión ni servicios externos.

Si necesitas hacer cálculos de interés compuesto con frecuencia, o simplemente quieres una herramienta que funcione sin complicaciones, este programa te sirve.

¿Para quién es útil?

Personas que ahorran o invierten y quieren entender el crecimiento de su dinero.

Estudiantes que necesitan practicar o ver resultados reales.

Desarrolladores o usuarios que prefieren trabajar en terminal.

Cualquiera que quiera una calculadora clara, rápida y confiable.
